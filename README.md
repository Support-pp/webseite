# Webseite

Support++ Webseite

https://support-pp.de

## Run it locally

1. [Install Hugo](https://gohugo.io/getting-started/installing/)
1. Open a command line and run:
   1. `git clone git@github.com:Support-pp/webseite.git`
   1. `hugo server`
1. Navigate to `http://localhost:1313/` in your browser
